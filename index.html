Multicast bandwidth savings: The results above reflect unicast (point-to-point) bandwidth, where a separate stream is sent to each receiver. With multicast — the default transport method for RAVENNA in professional media networks — a single stream is transmitted once on the network regardless of how many receivers are listening. This means network bandwidth consumption does not increase with the number of receivers, making multicast significantly more efficient for large-scale or broadcast deployments.
